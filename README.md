# Hospital Patient Analytics Dashboard

This project presents an interactive **Hospital Patient Analytics Dashboard** developed using **Microsoft Power BI**.
The dashboard analyzes healthcare data to provide insights into patient demographics, hospital operations, and financial performance.

---

## Tools & Technologies

* Microsoft Power BI
* Healthcare Dataset (Kaggle)
* Data Visualization & Analytics

---

## Dashboard Overview

The project contains **three analytical dashboard pages** designed to provide a complete overview of hospital data.

### 1. Hospital Overview

* Total Patients, Total Revenue, Average Age, and Average Stay Duration
* Monthly Patient Admission Trends
* Patient Gender Distribution
* Patients by Age Group
* Insurance Provider Analysis

### 2. Hospital Operations Analysis

* Top Hospitals by Patient Count
* Top Doctors by Patient Volume
* Admission Type Analysis (Elective, Emergency, Urgent)
* Blood Type Distribution

### 3. Financial & Treatment Insights

* Average Treatment Cost by Medical Condition
* Revenue by Admission Type
* Medication Usage Analysis
* Test Results Distribution

---

## Key Insights

* Identifies trends in **patient admissions over time**
* Analyzes **hospital and doctor performance**
* Shows **treatment costs and revenue patterns**
* Provides insights into **patient demographics and medical conditions**
